Add a new webpage to NB

This is an experimental feature... NB can be used to comment simple HTML pages.
  • Use this form to indicate the page that you'd like to comment with NB
  • That page needs to contain our embedded NB viewer. To do so, you need to insert the following snippet into the <head> section of your page:
    <script src='HOSTNAME/embed_NB.js' type='text/javascript'> </script>
  • If you can't add the embedded NB viewer for any reason (for instance, you don't own the page), you can use the following bookmarklet to insert the script. Note that you need to communicate this bookmarklet to your users (and explain them how to use bookmarklets...): 
    javascript:c=document.createElement("script");c.setAttribute("type", "text/javascript");c.setAttribute("src", "HOSTNAME/embed_NB.js");document.getElementsByTagName("head")[0].appendChild(c);
